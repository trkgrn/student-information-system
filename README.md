### Used Dependencies
* Backend
  * Spring
    * Spring Boot
        * Spring Security
        * JWT
        * Spring Web
        * Spring Data
            * Spring Data JPA
            * Spring Data Redis
        * Jedis (Redis Client)
        * Lombok
        * DevTools
        * ModelMapper
        * Flyway Core
        * Swagger UI
* Frontend
  * Angular
    * Angular Core
    * Angular Fire
    * Firebase (Cloud Storage)
    * AdminLTE 3
    * Sweet Alert 2
    * uuid
* Database
  * PostgreSQL
  * Redis
